### Touches :
- Entrer = faire spawn un joueur
- touches directionnelles = avances à gauche ou droite
- espace = sauter OU tirer
- X = sortir l'arme
- M = déclencher la mort subite
- clic = placer le joueur à l'endroit du clic (mode DEBUG)
